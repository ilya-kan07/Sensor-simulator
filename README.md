# Sensor Simulator

Этот проект — симулятор датчика, который генерирует значения, усредняет их за минуту и сохраняет в файл.

## Как запустить

1. Установите Python 3.6+.
2. Склонируйте репозиторий:
   ```bash
   git clone <URL>
   cd sensor-simulator
3. Запустите скрипт:
    ```bash
   python sensor_emulator.py
